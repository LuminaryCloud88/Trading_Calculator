# Trading Calculator

This program is used to quickly input values of a trade to help understand how much you will make or lose depending on the conditions.

## Description

This is my first project code built simply for me and my friends that trade. The goal of this project is to make working code with lots of features and to handle errors correctly. This is a work in progress and more features, as well as downloadable packaged programs will be available in the future. Please report and bugs so we can get started on fixing them right away.

## Getting Started

### Dependencies

* python3 must be installed for the program to execute as of 11-2025
* can be run on any operating system

### Installing

* You can download and run this code by:
1. Clicking the green Code button.
2. From the dropdown menu, select Download ZIP.
(The repository will be downloaded as a compressed .zip file to your computer. You will then need to extract the contents of this file.)

* No modifications are required.

### Executing program

1. Extract contents of zip folder
2. Navigate to the folder via terminal 
3. Run python3 main.py
